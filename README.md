## 🎯 Что делает проект

Проект реализует **три режима** работы:

### 1. 📊 Режим ввода матриц
Пользователь вводит две матрицы, сервер вычисляет их **свёртку** и показывает результат в виде таблицы. Доступны параметры:
- **Padding** (отступы) — добавление нулей по краям матрицы
- **Stride** (шаг) — насколько далеко сдвигается ядро

### 2. 🖼️ Режим обработки изображений
Пользователь загружает изображение и выбирает фильтр (ядро):
- 🔍 Без изменений
- ✏️ Выделение границ
- 📷 Размытие
- 🔪 Резкость
- 🎨 Сильные границы
- 😊 Гаусс (размытие)

Также можно ввести **своё ядро** вручную.

### 3. 🏗️ Режим конструктора CNN
Пользователь может создать **многослойную свёрточную сеть**:
- Добавлять/удалять слои
- Для каждого слоя задавать своё ядро
- **ReLU активация** применяется автоматически между слоями
- Результат показывает оригинал и изображение **после каждого слоя**

## 🧠 Что такое свёртка?

Свёртка (convolution) — это математическая операция, при которой **ядро** (небольшая матрица) скользит по **исходной матрице** (или изображению), вычисляя сумму произведений перекрывающихся элементов.

**Применение:**
- Обработка изображений (размытие, резкость, выделение границ)
- Свёрточные нейронные сети (CNN) — основа компьютерного зрения
- Распознавание лиц (Face ID)
- Фильтры в Instagram, TikTok
